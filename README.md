# 🎌 Otaku Flow: The Ultimate Anime Discovery Portal

**Otaku Flow** is a high-end, cinematic web application designed to help users navigate the vast ocean of anime and find their next masterpiece. Inspired by the sharp precision of Zoro's Three-Sword Style, it combines immersive visuals with smart filtering technology.

---

## 📌 Description

Finding the right anime shouldn't feel like being lost at sea. **Otaku Flow** solves this by offering a curated, interactive experience that guides you from a single choice to a full-blown binge session.

- ⚔️ **The Discovery Journey**: A 3-step state machine that filters by Genre, Mood, and Watch Style.
- 🧭 **Zoro's Compass**: Feeling lost? Let Zoro find a high-quality random masterpiece for you.
- 🎬 **Cinematic Landing**: Immerse yourself instantly with the auto-playing Elbaf Arc video banner.
- ❤️ **The Collection**: Save your favorites to your personal "Dojo" via LocalStorage.

---

## 🚀 Key Features

- 📊 **Dynamic Filter Hubs**
  - **Newbie Hub**: Perfect entry points for beginners.
  - **Heat**: The most popular trending shows right now.
  - **Binge Zone**: Epic sagas for the long-distance viewer.
  
- 🎭 **Interactive Discovery**
  - Use the "Select Path" feature to find anime based on how you feel and how much time you have.

- 🔍 **Global Search**
  - Real-time search across the entire Jikan database.

- 🌙 **Dual Theme Experience**
  - Switch between **Deep Cosmos (Dark)** and **Emerald Zen (Light)** modes instantly.

- 📖 **Premium Flip-Cards**
  - Hover over any card for a 3D flip effect showing detailed synopses, episode counts, and save options.

---

## 🌐 API & Tech Stack

- **Data Engine**: [Jikan API](https://docs.api.jikan.moe) (Unofficial MyAnimeList API).
- **Core**: Vanilla JS (ES6+), HTML5.
- **Styling**: Vanilla CSS3 (Custom Design System, Glassmorphism, 3D Transforms).
- **Storage**: Browser LocalStorage for user preferences and favorites.

## 🛠️ Installation & Setup

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/your-username/otaku-flow-portal.git
```

### 2️⃣ Launch the Portal
If you're using VS Code:
1. Install the **Live Server** extension.
2. Right-click on `index.html`.
3. Select **"Open with Live Server"**.
